Folder t0 - t4 contain five different trails. Each trail folder contains the following files:


1. ...mini40.csv is the csv file containing readings from force sensor, the order of the 
data in column is: time(ms), Fx(lbf), Fy(lbf), Fz(lbf), Tx(ibf-in), Ty(lbf-in), Tz(lbf-in)

2. ...mocap.csv is the csv file containing readings from all the nine optical trackers, the 
order of the data in column is: time(ms), index of the optical tracker, x, y and z position 
of the corresponding optical tracker(m).

Note: All the data are raw data. In order to visualize it, synchronization and smoothing might 
be needed.
